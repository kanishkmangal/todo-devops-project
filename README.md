# Simple To-Do List Application

This is a simple, clean, and beginner-friendly To-Do List web application built with React and Vite.

## Features
- Add new tasks
- Mark tasks as completed (strikes through the text)
- Delete tasks
- Clean and minimal UI

## Project Setup Instructions

1. **Install Dependencies**
   Navigate to the project directory and install the required packages:
   ```bash
   npm install
   ```

2. **Run the Development Server**
   Start the app locally:
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Open the URL provided in your terminal (usually `http://localhost:5174`) to view the app.

## Folder Structure
- `src/` - Contains the React source code.
  - `components/` - Reusable UI components.
    - `TodoList.jsx` - Component representing the whole list and input form.
    - `TodoItem.jsx` - Component representing an individual task.
  - `App.jsx` - Main application component.
  - `index.css` - Global styles and minimal CSS.
